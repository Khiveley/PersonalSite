KensSellerListings - JPACRUDProject

A simple CRUD project on a single table database called "KensSellerListings." I created a database based on Pokemon cards that I collected and looked to sell.

Technologies: Java, Apache Tomcat, MySQL Workbench, JPA, Spring Boot

Helping Hand Full Midterm Project

My group and I created a site "HelpingHand" to help fight hunger in through creating an app that allows users to create appointments for pickup of food at a food pantry. Pantries can also post their inventory. This also helps to ensure that front line workers can have improved safety from less time in contact.

Technologies/Methodologies: Spring Boot, Kanban board (Trello), MySQL Workbench, Figma (Wireframes),JPQL, Git Branching, Agile, HTML 5, CSS, Bootstrap

After Boot - Study Session Event Tracker

I created an event tracker called, "AfterBoot," which was designed as a study log to track my time in reference sources by language. My ambitions are to hone my Java, JavaScript, and SQL skills as well as learn Python and possibly Unity/C#.

Technologies: Spring Tool Suite (Spring Data JPA, Spring Boot), MySQL Workbench (SQL), Atom, Gradle, Postman, Java, MAMP, Bash Terminal, VS Code, XMLHttpRequests, JavaScript

Borrow It - Entertainment Lending Full Stack Final Project

My group and I created a site "BorrowIt,"to stave off boredom during quarantine and allow users to borrow from one another various forms of entertainment: books, movies, games, videogames, etc.

Technologies/Methodologies: Angular, Spring Boot, Spring Security, Gradle, TypeScript, Java, JavaScript, HTML 5, CSS, TDD, MySQL Workbench (SQL), Agile, Wireframes, Kanban Board (Trello)
